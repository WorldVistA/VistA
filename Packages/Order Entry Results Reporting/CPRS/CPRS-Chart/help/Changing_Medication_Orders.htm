Changing Medication Orders

Users can only change Inpatient Medication orders for outpatients (IMO) if the patient is in an authorized IMO location.

To change a Medication order:

1.     Select either the Meds tab or the Orders tab.

2.     Select the medication order to select it.

3.     Select Action | Change... or right-click the order and click Change....

Note:      If the provider or location has not been defined, you will be prompted for that information.

 

4.     Complete the changes as appropriate in the dialog box that appears on the screen.

5.     Select Accept.

6.     You may sign the order now or later.
