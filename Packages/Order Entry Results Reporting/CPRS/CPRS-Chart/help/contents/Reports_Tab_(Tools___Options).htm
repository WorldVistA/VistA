Reports Tab (Tools | Options)

This tab allows you to set the date ranges and the maximum number of occurrences for CPRS reports. You can change the settings for all reports or for individual reports.

To change the date range and maximum number of occurrences for all CPRS reports:

1.     Select Tools | Options....

2.     Select the Reports tab.

3.     Press the Set All Reports... button.

The "Change default parameters for all CPRS reports" dialog displays.

 

4.     Change the value in the Start Date and Stop Date fields by selecting the appropriate field and by doing one of the following:

o     entering a date (e.g. 6/21/01 or June 21, 2001).

o     entering a date formula (e.g. t-200).

o     pressing the button to bring up a calendar.

 

5.     After you have entered a start and stop date, you can change the maximum number of occurrences (if necessary) by using the Max Occurrences field.

6.     Select OK.

A confirmation dialog box will appear.

 

7.     Select Yes to confirm and save your changes.

8.     Select OK to close the Options dialog box.

 

To change the date range and maximum number of occurrences for individual CPRS reports:

1.     Select Tools | Options...

2.     In the Options dialog, select the Reports tab.

3.     Select the Set Individual Report button.

The "Customize individual CPRS report parameters" dialog will appear.

4.     Place the cursor in the "Type the first few letters of the report you are looking for:"field (located at the top of the dialog box) and type the name of the report that you would like to change

-or-

use the scroll bars to find the report.

 

5.     Change the value in the Start Date and/or Stop Date field by clicking in the appropriate column and doing one of the following:

o     entering a date (e.g. 6/21/01 or June 21, 2001).

o     entering a date formula (e.g. t-200).

o     pressing the button to bring up a calendar.

 

6.     After you have entered a start and stop date, you can change the maximum number of occurrences (if necessary) by using in the Max field.

7.     Select Apply to save your changes

-or-

select OK to save your changes and close the dialog box.

 

8.     Select OK to close the "Options" dialog box.
