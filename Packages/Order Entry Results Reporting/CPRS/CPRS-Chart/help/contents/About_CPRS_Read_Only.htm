About CPRS Read-Only

CPRS Read-Only enables you to select a patient and view reports from the patient's medical record. As the name implies, you cannot enter, or update the information connected with any patient. The purpose of CPRS Read-Only is to give controlled access to specific users that will enable them to review the medical records but not change any information.

Users will be able to open the patient charts made available to them, including accessing remote data from other VA Medical Centers (VAMCs) facilities and from the Department of Defense if available. Read-Only users can view, copy, and print reports that are available from the Reports tab of CPRS.

Each site controls who has CPRS Read-Only access. To grant a user CPRS Read-Only access, someone at the site (probably whoever creates new users) will make entries into the NEW PERSON file to give the user access only to the Reports tab and usually assign a restricted patient list that contains the appropriate records. If the list is not assigned when you try to run the application, CPRS Read-Only will show an error message that states "Patient selection list not found." When the user click OK to close the message box, CPRS Read-Only aborts.
