Graphing: Choosing Display Items on the Select Items and Define Views Dialog

CPRS graphing enables you to display graphs directly from the Select Items and Define Views dialog box. Using this dialog box, you can specify the following display options:

Top

Select this option to display your graph in a single-pane view, or to display it at the top of a split view if the Split View check box on the main window is selected.

Bottom

Select this option to display your graph on the bottom pane of a split view.

Both

Select this option to display your graph in both the top and bottom panes of a split view.

No Change

Select this option if you want to use the display settings you have selected in the main window.
