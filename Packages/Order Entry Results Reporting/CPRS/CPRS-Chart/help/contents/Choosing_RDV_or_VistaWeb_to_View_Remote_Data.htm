Choosing RDV or VistaWeb to View Remote Data

Previously, a user would set their preference to use either VistaWeb or Remote Data Views. This was set once and then stayed until the user changed the preference. This changed in CPRS v.27 so that both ways of retrieving remote data are now available through buttons that are below the menu bar: VistaWeb and Remote Data.

Because users choose their method of viewing remote data at the time they go to find the data, there is no longer a way to permanently set the remote data method using the Tools | Options...  menu item. The tab now only has a description of how to select the option.
