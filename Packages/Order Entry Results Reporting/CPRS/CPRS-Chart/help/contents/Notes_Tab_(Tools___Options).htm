Notes Tab (Tools | Options)

Notes...

This option on the Notes tab allows you to configure defaults for editing and saving notes. Use the selection arrows to change the number of seconds between auto save intervals for notes. You may also assign a default cosigner for notes by selecting the drop-down button and selecting a provider. Y o u may also click on the either of the two check boxes, if you wish to be prompted for a subject for progress notes and if you wish to verify note titles.

Document Titles...

You may select a personal list of document titles to be displayed for several different types of documents. Select on the drop-down button on the Document class field and select the class of document for which you would like to create a list. When you have selected a document class, the Document titles field is automatically populated with all available choices. Highlight one and select Add . Hold down the Control key to select more than one title at a time. To select a title from your list as your default, highlight it and select on Set as Default . Select on Save Changes if you will be making more changes on this dialog before you select OK .
