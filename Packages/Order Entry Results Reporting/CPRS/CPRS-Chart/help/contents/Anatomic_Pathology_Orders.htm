Anatomic Pathology Orders

Note:      The Anatomic Pathology order dialog is not currently available, but will be available soon. Sites will be notified when this feature becomes available.

Anatomic Pathology (AP) laboratory service now has a mechanism for clinicians to provide required patient-specific,procedure-specific, and specimen-specific information to facilitate specimen processing by pathologists.  A new Anatomic Pathology Order Dialog is available with the following available order types:

  • Bone Marrow

  • Bronchial Biopsy

  • Bronchial Cytology

  • Dermatology

  • Fine Needle Aspirate

  • Gastrointestinal Endoscopy

  • General Fluid

  • Gynecology (Pap Smear)

  • Renal Biopsy

  • Tissue Exam

  • Urine

  • Urology, Bladder/Ureter

  • Urology, Prostate

When opening the Anatomic Pathology order dialog, you must first select one of the available order types, such as seen below, for example Bone Marrow, Bronchial Biopsy, etc. 


Because each order type may require different information, different order dialog fields display when the user selects an order type.  

 

The Anatomic Pathology Ordering dialog is shown below. On this dialog, the user selects the type, such as Dermatology. Red boxes and numbers added to describe different areas.

 

 

The dialog has several areas that interact. Some areas of the dialog are described briefly below.

  1. Order-specific fields such as Urgency and Collection Date/Time.
  2. List of Specimens for this order.
  3. The Current Specimen being viewed in box 4.
  4. Specimen-specific fields, such as the Specimen Description//Anatomic Site.
  5. Button to delete the current specimen.
  6. Multiple tabs to enter free text fields, such as Clinical History.

 

When the user selects Anatomic Pathology from an order menu or the Orders pane, the Anatomic Pathology Ordering dialog displays and the first thing to select is an order type (e.g., Bone Marrow, Dermatology, etc).

 

When an order type is selected, the rest of the fields on the dialog change to gather the information for that type of order. 

 

To place an Anatomic Pathology order, use these steps:

  • Select Anatomic Pathology from the Orders menu.
  • In the Anatomic Pathology Ordering dialog, under Available Orders, select the type of order (e.g., Bone Marrow, Dermatology, etc.).
  • Fill out the order specific information, such as Urgency, Collection Date/Time, Collection Type, etc.
  • On the right of the dialog, select a specimen from the drop down list and provide a specimen description. There must be at least one specimen and a specimen description to accept the order. As you select a specimen and description, a list of specimens is created. To view each specimen, highlight the specimen from the list and the description will display. 
  • The fields may change when you select a type. Fill out the necessary fields.
  • For some order types, the user can enter some free text under Clinical History on such tabs as Pre-Operative Diagnosis, Operative Findings, and Post-Operative Findings.
  • At the bottom of the dialog, the order text created when the user filled in the fields is displayed.
