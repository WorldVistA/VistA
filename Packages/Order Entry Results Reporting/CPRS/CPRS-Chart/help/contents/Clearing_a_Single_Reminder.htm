Clearing a Single Reminder

You will probably process several reminders for a single visit. If you have entered information on a reminder, but you need to start over on that reminder only, you can simply click Clear. This removes the entries you have and lets you start again on this one reminder.

Note:      Selecting Clear will remove the information from only one reminder. Be careful that you are on the correct reminder before you select Clear.
