New Features and Components of CPRS

CPRS offers the following new features and components that OE/RR did not have:

·      Order checking

·      Adverse Reaction Tracking

·      Discharge summaries

·      Problem list

·      Lexicon

·      Order sets

·      Time-delay orders

·      Graphical user interface (GUI)

·      HL-7 communications protocol for transferring messages among packages

·      Text Integration Utilities (TIU) and Authorization/Subscription Utility (ASU), which will allow setting of business rules, document types, user classes, etc. to control access to documents, reports, verification, signing, and so on

·      Support of the Ambulatory Care Date Capture project for capturing encounter data for workload credit, as well as capturing clinically relevant data for reminders and reports
