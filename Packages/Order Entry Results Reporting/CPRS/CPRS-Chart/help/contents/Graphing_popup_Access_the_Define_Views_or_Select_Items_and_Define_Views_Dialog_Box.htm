Graphing: Access the Define Views or Select Items and Define Views Dialog Box

·      On the CPRS main menu, select Tools and then select Options. CPRS displays the Options dialog box.

a)    On the Graphs tab, select View Definitions. CPRS displays the Define Views dialog box.

-or-

·      Select Select/Define on the right-click menu. CPRS displays the Select Items and Define Views dialog box.

-or-

·      Select Select/Define at the bottom of the graphing window. CPRS displays the Select Items and Define Views dialog box.

-or-

·      On the CPRS Reports tab, select Graphing (local only) in the Available Reports list. CPRS displays embedded graphing functionality.

a)    Select the Select/Define button. CPRS displays the Select Items and Define Views dialog box.
