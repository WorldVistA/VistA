Removing a Surrogate

To remove a surrogate, follow these steps: 

  1. Select Tools | Options.
  2. Select the Surrogates tab.
  3. Highlight the surrogate to be removed.
  4. Select Remove Surrogate .
  5. On the Remove Surrogate popup, select OK.
  6. On the Surrogates Management screen, select Apply to save your changes.
  7. Select OK to close the Options window.
