Active Medications Display

CPRS displays the patient’s active medications in a list on the Cover Sheet tab. For most inpatients, the list contains

  • Inpatient medications

  • Clinic medications

However, if the ward location for the inpatient’s ward has a service of Domiciliary, the list contains

·           Inpatient medications

·          Clinic medications

·         Outpatient medications

 

CPRS users can select a medication from the active medications list to view detailed information about the medication.
