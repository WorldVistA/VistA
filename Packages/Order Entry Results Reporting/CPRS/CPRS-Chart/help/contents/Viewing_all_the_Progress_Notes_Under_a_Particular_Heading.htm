Viewing All the Progress Notes under a Heading

To view all the progress notes under a particular heading, follow these steps:

1.     Select the Notes tab.

2.     Double-click the heading that you would like to view.

The notes that are related to that heading will appear in a table on the right side of the screen.

 

3.     To view the details of a specific note, select the note from the table. You can also sort the table by clicking on the column you wish to sort by (click the column again to sort the table in inverse order).
