Placing a Quick Order

To place a quick order, do the following:

1.     Select the Orders tab.

2.     Select the item for Quick Orders.

3.     If necessary, select a category.

4.     Select the Quick Order.

5.     When finished, select Done .

Note:        When you have finished placing orders, you can sign the orders now or later.
