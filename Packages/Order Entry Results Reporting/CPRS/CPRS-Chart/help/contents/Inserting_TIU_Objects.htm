Inserting TIU Objects

To insert a TIU object, use these steps:

1.      Right-click in the Template Boilerplate area.

2.      Select Insert TIU Object.

3.      Locate the object you want (you can either scroll or type in the first few letters) and select it.

 

You can only select one object at a time. The object will be placed in the template where the cursor is.
