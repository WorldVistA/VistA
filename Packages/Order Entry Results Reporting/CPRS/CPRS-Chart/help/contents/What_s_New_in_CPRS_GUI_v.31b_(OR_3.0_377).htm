What's New in CPRS GUI v31b (OR*3.0*377)

CPRS v31b is a large release that incorporates changes to many parts of CPRS. Changes are listed below:

Support for Women's Health

  • Changes to support the System for Mammogram Results Tracking (SMART) . See SMART Features.
  • Support for alerts about Potentially Unsafe Medications or Exposures (Teratogens). This project is sometimes referred to as T-drugs.

 

Cover Sheet

  • Customize which Cover Sheet Panes display and the order in which they display

 

Notifications

  • Long text alerts
  • Patient-centric Notifications
  • SMART notifications

 

Other than Honorable (OTH) Care Periods

  • OTH button

 

Veteran Health Library

 

Consults

  • Locking Consult Templates

 

Dietetics

  • Prevent Previous Active Diet Replacing NPO Order Upon Expire, Discontinue, or Cancel

 

Medication Orders

  • The EXCEPT Conjunction is no longer allowed in complex doses
  • For some medications, longer than 90-day supplies may now be ordered

 

Auditing Copy/Paste

  • Overview of Copy/Paste Information
  • Setting How Pasted Text Will Display
