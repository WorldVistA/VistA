Graphing: Renaming Predefined Views

Take the following steps to rename a predefined view:

1.     Access the Select Items and Define Views or Select/Define dialog box using one of these methods.

2.     In the Source list, select the name of the view you want to rename.

3.     Select Rename.

CPRS displays the Rename your Personal View or Rename this Public View dialog box.

 

4.     Type a new name for your personal or public view.

5.     Select OK.

CPRS displays the Confirm dialog box.

 

6.     To change the name of your view, select Yes.
