What's New in CPRS GUI v32a (OR*3.0*539)

CPRS v32a includes the following changes:

 

Allergies: Notification about Drug Class cross-checking

  • Allergies from the Cover Sheet
  • Allergies Entered from the Orders Tab

 

Anatomic Pathology Ordering Dialog

  • New dialog helps users enter Anatomic Pathology orders

 

Encounter Provider Selection (similar names)

To better address selecting a name for various purposes, such as an Encounter or additional signers, CPRS will display a dialog when similar names are in the system to help the user select the correct name.

  • Selecting an Encounter Provider

 

Non-VA Medications

Added "(Documentation)" to Non-VA Medication labels. Now it reads, "Non-VA Medications (Documentation)"

  • Non-VA Medications (Overview)
  • Documenting Non-VA Medication Informations

 

Reports

  • Added reports

 

Surrogates

  • Surrogates Tab 
  • Adding a a Surrogate
  • Editing a Surrogate
  • Removing a Surrogate

 

TIU Changes

  • Template Required Fields:  Identify and complete
  • Improved Progress Notes Custom View, Show More

 

Unflagging Orders

  • New flagging orders features
  • Unflagging order can now be limited to users with specific security keys

 

Unified Action Profile (UAP)/Medication Reconciliation

  • Unified Action Profile (UAP) Provides a View where Inpatient and Outpatient Medications are combined

 

VBECS

  • VBECS Order Dialog change with Date for All Components
