Exclude from Group Template

The Exclude from Group Template feature is available only for templates within group templates.

Exclude from Group Template lets you designate a template within a group template that should not automatically be placed in a document when you insert the group template into a document.

The excluded templates is still visible in the tree view, and you can place them in the document individually.

Exclude from Group Template might be useful if you have exams that are sometimes, but not always done with a visits. For example, you could insert the group template and then insert any excluded templates if those exams or activities were also conducted.
