Flagging an Order

With CPRS, you can flag an order to draw attention to it. When an order is flagged, the word "Flagged" will display in the Orders column and a red box will display in the Service or Event column.

The order will remain flagged until someone "unflags" the order or if the time elapses for the flag to expire. Sites can set the length of time for the flag to expire. CPRS records the name of the person who flagged the order and the date and time that it was flagged.

When the user processes the order, CPRS displays the reason for the flag in the order text on the Orders tab.

To flag an order, use these steps:

1.   Select the Orders tab.

2.   Select the Order that you would like to flag.

3.   Select Action | Flag ....

The Flag Order dialog displays.

The Flag Order dialog with the red rectangle added showing that multiple providers can be added as recipients.

4.   Enter a reason for the flag in the Reason for Flag field. Users can choose a reason from the drop-down list, choose a reason and add additional text, or enter their own text. The Reason for flag field has an 240-character limit.

5.   Add recipients by highlighting names and selecting Add. If someone is added by mistake, highlight the name in the right box and select Remove or you can Remove All.

Note:   A recipient must have the FLAG ORDER FOR CLARIFICATION notification/alert enabled in order to receive the alert.

 

6.   If wanted, set a No Action Alert date by selecting the check box and the setting a date using either the box with three dots (the ellipsis) .

7.   Select Flag,  when ready.

 

Note:   If the OR FLAGGED & WARD COMMENTS parameter is turned On, flagged Order comments and Ward comments will display directly in the Order column. This parameter is turned Off by default; a CAC must activate it for your site. Refer to the CPRS Technical Manual: GUI Version for information on this parameter.  

Note:   If the provider has an NPI, it will not display on the CPRS Orders tab.

 

New Flag Order Notifications

Two additional notifications have alert users if the flag expires of if comments are added the flag.

An example of the two new flagged notifications are shown within the red outline. Comment Added to Flagged Order and Order flag expired are both shown.

 

If comments are added, they are displayed in the flag details.

To add a comment, users can either highlight the flagged orders and choose Action | Flag Comment or right-click on the order and select Flag Comment.
