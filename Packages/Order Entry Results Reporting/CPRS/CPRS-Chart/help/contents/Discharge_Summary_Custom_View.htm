Discharge Summary: Custom View

You can set up the list of discharge summary the way you like it. You get the same sorting choices as on the View menu, but you can further refine the search.

For Unsigned Documents, Uncosigned Documents, and Signed Documents/Author, you can select

· author

· date range

· maximum number of summaries CPRS should display

· the sort order (ascending or descending)

 

For Signed Documents (All) and Signed Documents/Date Range, you can select

· date range

· maximum number of summaries CPRS should display

· the sort order (ascending or descending)
