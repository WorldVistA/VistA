Grammar Checking for Notes, Consults, and Discharge Summaries

You can use grammar checking on notes, consults, and discharge summaries if you have Microsoft Word.

To check a document or a selection, use these steps:

1. To check the grammar of an entire document, skip to step 2. Click-and-drag to select text if you want to only check a selection.

2. Right-click on the selection or document and select Grammar Check... .
