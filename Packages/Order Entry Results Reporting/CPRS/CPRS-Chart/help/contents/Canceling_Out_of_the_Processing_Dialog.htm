Canceling Out of the Processing Dialog

If you reach the Reminders processing dialog by mistake or if you have entered information for reminders, but you need to get rid of that information and start over, simply click Cancel.

Note:       If you click Cancel, you will lose all of the information for reminders that you have entered . Use this option only if losing this information is not important.
