Manually Evaluating Reminders

If you open CPRS to a tab other than the Cover Sheet, you will have to start the reminder evaluation process manually.

To manually begin the evaluation of reminders, use these steps:

1.     Select the Reminders button , which is available from any tab. If the reminders button has a clock on it, the reminders have already been evaluated.

-or-

select the Reminders Drawer on the Notes tab.

When the evaluation begins, the reminders button will show an animated magnifying glass.

 

2.     Once Reminders have been evaluated, the Available Reminders dialog opens.

Note:       To assist users in remembering the meaning of each Reminder icon, click on View and note that a graphic of each icon appears next to the name of the icon. The icons with check marks indicate which types of reminders will be displayed on the Available Reminders dialog.
