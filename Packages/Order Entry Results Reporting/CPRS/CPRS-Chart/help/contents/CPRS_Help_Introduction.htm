Help Introduction

Welcome to the help system for the Computerized Patient Record System (CPRS). Please use the Contents or Index to the left to find the information you are looking for. If the Table of Contents or Index tab is not visible, please use the information below to display those tabs.

To find out more about the changes in CPRS v33SWD, please select the following:

What's New in CPRS GUI v33SWD (OR*3.0*608)

 

To find out more about how to use the Online Help system, please continue reading below.

You can use the Contents, Index, or Search tabs to look for the item you want to find.

  • The Contents tab has books and pages  (also known as topics). Books organize topics into a group. Select a page to take you to the help topic.

  • The Index is a list of terms you can read through and select to take you to a topic that contains that item or that concept.

  • The Search tab enables users to type in part or all of a word or words to locate in help topics. After the user types in text and either presses enter or selects the List Topics button.

 

Showing or Hiding the Contents, Index, and Search Tabs after Initial Display

After you have gotten the Help to display with the Contents, Index, and Search tabs, you can hide them to view the Help in the entire window and you can show the tabs again.

To hide the tabs, select the word "Hide" in the upper left corner of the Help window.

 

To display the tabs again, simply select the word "Show" in the same location.
