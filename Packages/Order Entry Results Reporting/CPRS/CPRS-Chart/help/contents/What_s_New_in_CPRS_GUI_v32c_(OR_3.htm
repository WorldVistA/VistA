What's New in CPRS v32c (OR*3.0*588)

CPRS v32c contains Write Access functionality, which enables an authorized VistA user to permit or restrict write access for each CPRS tab, and for specific functionality within each tab (for example, Cover Sheet, Vitals, Encounters and specific Ordering Display Groups). 

 

The VistA user can permit or restrict write access at the System, Division or User level. Write Access functionality eases the transition from CPRS VistA to the new Electonic Health Record (EHR) system. 

 

 CPRS v32c also repurposes the COVID-19 Identifier banner as a New EHR banner if you are in an organization that has migrated to a new EHR.
