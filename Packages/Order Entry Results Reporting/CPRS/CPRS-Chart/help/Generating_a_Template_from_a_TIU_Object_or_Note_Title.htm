Generating a Template

You can also generate a template from an existing Template Field or from a boilerplated note title.

To generate a template, use these steps:

1.     Select the appropriate tab: Notes, Consults, or D/C Summ.

2.     Select Options | Edit Templates for personal templates or Options | Edit Shared Templates for Coordinators working on shared templates.

3.     Select the template under which you want the new template.

4.     In the template editor, select Action | Generate Template.

5.     Select the source from which to generate the template: Patient Data (object) or Boilerplated Note Title.

6.     Locate the Patient Data or Note Title by either typing a few letters of the name or scrolling through the list. Then click the appropriate item and click OK.

Note:      When you click OK, the boilerplate text or patient data is brought into the template and a new template is created with either the object name or the note title.

 

7.     Add any necessary content to the document and click Apply to save the template or click OK to save the template and exit the editor.
