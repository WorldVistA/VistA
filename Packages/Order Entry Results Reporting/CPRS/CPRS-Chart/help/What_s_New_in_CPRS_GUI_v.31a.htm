What's New in CPRS GUI v.31a (OR*3.0*434)

  • Two-Factor Authentication - CPRS now uses two-factor authentication, which means that users will user their PIV card and PIN to log in to the application.

  • Return to Clinic - The feature enables providers to place an order to request that a clerk set up return or follow up appointments to a clinic.

  • Support for Joint Legacy Viewer - This release has support for when the Joint Legacy Viewer or JLV is released.

  • Consult Toolbox access in CPRS - The Consult Toolbox can now be accessed from CPRS if your site has done the proper set up. There is also a link to some of the documentation for the Consult Toolbox.

  • New Health Insurance and Health Plan Benefit Information on the Patient Inquiry dialog - Two new items are displayed in the Patient Inquiry dialog: the Permanent & Total Disabled status, and Health Benefit Plans.
