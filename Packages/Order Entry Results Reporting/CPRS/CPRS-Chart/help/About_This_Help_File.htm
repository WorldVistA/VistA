About this Help File

This help file is for patch OR*3.0*434 (CPRS GUI v.31a). There was a major change in this help file a few versions back. Due to technical changes, it was necessary to remove many of the graphics in this help file. With the graphics in, the file size became unmanageably large. However, many of the graphics that were in the help file are included in the CPRS User Guide: GUI Version. Your site should have this document. If you do not have it, please contact your Information Resource Management (IRM) staff or your Clinical Application Coordinator (CAC).

The help file system is now a system of HTML files.
