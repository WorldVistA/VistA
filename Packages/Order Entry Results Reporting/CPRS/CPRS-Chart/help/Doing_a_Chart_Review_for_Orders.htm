Doing a Chart Review for Orders

You can do both a nurse verification of orders and a chart review through CPRS.

When a nurse does the chart review, the nurse's initials are then placed in the appropriate column on the Orders tab.

Note:      If you do not have the correct key, you will not see the Chart review option on the Action menu.

 

To do a chart review for orders, use these steps:

1.     Select the Orders tab.

2.     Hold down the CTRL key and click the orders you want to sign as reviewed.

3.     Select Action | Chart review.

4.     In the dialog that appears, enter your electronic signature code and press Enter.
