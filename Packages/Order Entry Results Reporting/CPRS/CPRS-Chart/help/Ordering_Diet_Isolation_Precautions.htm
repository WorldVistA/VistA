Ordering Diet Isolation/Precautions

To place a diet isolation or precaution order, follow these steps:

1.          Select the Orders tab.

2.     Select the active orders view from the View Orders pane

-or-

select View | Active Orders (includes pending, recent activity).

Note:      The Encounter Information dialog appears if no encounter information has been entered.

 

3.     Select the Isolation/Precaution tab.

4.     Select the type of precaution (Protective, Respiratory, Strict, or Wound/Skin).

5.     Type the instructions in the Instructions text field.

6.     Select Accept Order.

7.     When finished, select Quit.

Note: The order must be signed before it is sent. You can either sign the order now or wait until later.
