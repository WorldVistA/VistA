Reminder Evaluation for a Patient

When you open a patient record, if your default tab is the Cover Sheet, an evaluation is performed to determine which reminders defined in a CPRS parameter apply to the selected patient. If you have set CPRS to open to a different tab, you will have to begin the evaluation process manually (see Manually Evaluating Reminders).

While the evaluation is proceeding, you will see an animated magnifying glass over the Reminders button. When the evaluation is complete, the magnifying glass is no longer visible.

When this evaluation is complete, you can get additional information about the reminder and how it can be resolved.

Right-click on a reminder either in the tree view or on the cover sheet to bring up a popup menu with these options:

路      Clinical Maintenance - shows the possible resolutions and the findings associated with the reminder

路      Education topic definition - may list the education topics that have been defined for a reminder. You can select a topic to view the desired education outcome and any standards.

路      Reminder Inquiry - will show the reminder definition describing which patients are selected for this reminder

路      Reference Information - which can list web sites with additional information.

 

Each of these options will bring up a window. When you are finished with the window, click Close.
