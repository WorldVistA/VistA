Icon Legend

This non-modal, stay-on-top window has five tabs that define various icons for items in CPRS: templates, reminders, notes, consults, and surgery. The window can be opened using the following methods:

路      From the Notes, Consults or D/C Summ tab, select View | Icon Legend.

路      Right-click in the Templates Drawer.

路      From the Templates Editor, select Tools | Icon Legend.

路      Right-click anywhere a Reminder is displayed including the Coversheet, Reminders Drawer, and the Reminders Tree available from the Reminder Button.
