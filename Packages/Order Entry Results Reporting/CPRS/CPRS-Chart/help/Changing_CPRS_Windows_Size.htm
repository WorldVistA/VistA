Changing CPRS Windows Size

CPRS uses standard Windows method for resizing windows. In addition, CPRS allows you to change the size of some data fields within CPRS.

路      To change the width of the window, you position the mouse pointer over the right or left edge (when positioned correctly the pointer will become a two-headed arrow), and click and drag to make the window narrower or wider.

路      To change the height of the window, you position the mouse pointer over the top or bottom edge (when positioned correctly the pointer will become a two-headed arrow), and drag up or down to make the window taller or shorter

路      To change the width and height simultaneously, you position the mouse pointer over the bottom-right corner and click and drag diagonally to increase or decrease the overall size of the window

路      To change the size of adjustable field on a tab or window, you position the mouse pointer over the vertical panel divider and drag left or right to increase or decrease the size of either panel. For Example, in the window below, the left-hand panel which lists All Consults is too narrow to show all the information about each Consult.
