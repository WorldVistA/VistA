Finding Specific Text in a Document

To find specific text in a document, follow these steps:

1.     Select the appropriate tab: Notes, D/C Summ, Consults, or Surgery.

2.     Select a document title.

3.     Right-click in the details pane where the text of the report is displayed.

4.     Select Find in Selected kind of document (such as note, summary, or report).

5.     Type in the text you wish to find in the Find what field of the Find dialog box.

6.     To narrow your selection, use these steps.

7.     Select Find Next.

The appropriate text will be highlighted if it is found in the document.

 

8.     To find another instance, click or select Find Next again.

9.     When finished, close the dialog.
