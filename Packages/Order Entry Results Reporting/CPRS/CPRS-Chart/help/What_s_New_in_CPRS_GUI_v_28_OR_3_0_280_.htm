Order Checking

·       Site-Defined Clinical Reminder Order Checks were added to CPRS. This allows sites to define their own order checks by defining Clinical Reminders terms and rules, and then the order check text that should display to the user when the term is present or the rule is met.

·       Local Order Checks have some new order checks and new dialogs.

Patient Record Flags

· The Patient Record Flags dialog was updated. The number of items is listed after the Category label for each category.

 

Primary Care Button

·      Associate Provider has been added to the display when the Primary Care button is selected.

 

Printing in CPRS

·      CPRS now uses the same printer you printed to last until you change it or log our and log back in. On log out, it will revert to the default printer.

 

Tools

·      CPRS Tools number of items increased and ability to use submenus

·     The VistaWeb button will now change colors, displaying blue text on the button when remote data is available.
