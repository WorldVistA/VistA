CPRS Help Introduction

Welcome to the help system for the Computerized Patient Record System (CPRS). Please use the Contents or Index to the left to find the information you are looking for. If the Table of Contents or Index tab is not visible, please use the information below to display those tabs.

CPRS is now using a new help format that is based on Hypertext Markup Language (HTML). The resulting help files might be slightly different than what you are accustomed to in the help file. However, the basic functions remain the same with the exception of the search tab, which is now a simple listing of items or phrasing used in the help.

To find out more about the changes in CPRS v31a, please select the following:

What's New in CPRS v31b?

 

To find out more about how to use the new help system, please continue reading below.

You can use the Contents, Index, or Search tabs to look for the item you want to find.

  • The Contents tab has books and pages. Books are simply category markers and cannot be selected. Select a page to take you to the help topic.

  • The Index is a list of terms you can read through and select to take you to a topic that contains that item or that concept.

  • The Search tab is a listing of words, letters, or characters that occur in topics. It is generated automatically by the help software and may not be as helpful as the other two tabs for finding the information that you need.

Can't See the Contents, Index, or Search Tabs?

Because the new help format is based on HTML, there are several difference in how the system works. One difference is that the system uses ActiveX controls to display the Contents, Index, and Search tabs on the left of the main help screen. Some workstations may block the tabs initially. If the tabs are blocked, the browser will display something like the following:

 

 

To display the tabs, use the following steps:

  1.  If it is not already up, launch Internet Explorer.

  2.  Select Tools | Internet Options.

  3. Select the Advanced tab.

  4. Under the Security category, ensure that the Allow active content to run in files on MyComputer is checked.

  5. Select OK.

Showing or Hiding the Contents, Index, and Search Tabs after Initial Display

After you have gotten the Help to display with the Contents, Index, and Search tabs, you can hide them to view the Help in the entire window and you can show the tabs again.

To hide the tabs, select the word "Hide" in the upper right corner of the Help window as shown below.

 

To display the tabs again, simply select the word "Show" in the same location.

 

Using the Contents, Index, and Search Tabs

The Contents, Index, and Search tabs are a collection of links to different help topics. The Contents is a listing of topics grouped by subject. The Index is a listing of various terms that a user might be looking for, and finally, the Search tab contains a listing of most terms in the topics that is created by the help program itself. Hovering over a term will display the topic name, term, or number displays the topic that the link will take you to.

  • Contents Tab: To find a topic in the Contents tab, simply look through the topic until you find the category and subject that you need information on.
  • Index Tab: To find an item on the Index tab, use the letters across the top of the tab to search for terms beginning with the letter you select. Then look for the term that you need information on.
  • Search Tab: To find a term on the Search tab, use the letters across the top of the tab to search for terms beginning with the letter you select. Then look for the term that you need information on. If a term has more than one entry, hover over the number after the term to show which topic it will take you to.
