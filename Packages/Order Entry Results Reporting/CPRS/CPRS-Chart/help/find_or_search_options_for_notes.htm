Find Options

To help more narrowly define a search, you can set two find options:

·      Match Case

·      Whole Words Only.

 

If you check Match Case, you can limit your search to find only words that have the same capitalization that you enter. For example, you could find "Patient" without having to look at all the titles with the word "patient."

If you check Whole Words Only, you can locate titles that contain specific words. For example, this search would allow you to find "patient" without wading through all of the titles with the word "patients."
