Verifying a Problem

To verify a problem on a patient's problem list, use these steps:

1.     Select the Problems tab.

2.     Select the problem in the problem list.

Note:      If you try to select a problem that has an inactive diagnosis or procedure code, CPRS prompts you to select a problem with an active code.

 

3.     Select Action | Verify

-or-

right-click the problem and select Verify on the pop up menu.
