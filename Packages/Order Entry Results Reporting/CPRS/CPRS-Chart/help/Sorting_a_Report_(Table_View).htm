Sorting a Report (Table View)

If a report is available in a table view, the table can be sorted alphabetically, numerically, or by date.

To sort data in a report table:

1.     Select the column heading you wish to sort by.

The table will be sorted alphabetically (A-Z), numerically (0-9), or by date (most recent-least recent).

2.     If you click the column heading again, the table will be sorted in inverse order (Z-A, 9-0, or least recent-most recent).

3.     To perform a secondary sort, click on another column heading.

Note:     If you hold the pointer over the table, a hover hint will appear indicating the criteria used to sort the table.
