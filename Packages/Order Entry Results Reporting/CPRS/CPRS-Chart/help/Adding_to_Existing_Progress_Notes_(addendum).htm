Adding to Existing Progress Notes (addendum)

CPRS enables you to make addenda to Progress Notes. After reviewing a Progress Note, you may want to add comments to the note.

You can create multiple addenda to a single Progress Note if you choose.

To place an addendum to a Progress Note, follow the steps below:

1. Click the Notes tab.

2. From the index of Progress Notes on the left of the dialog, click the Note to which you want to add an addendum.

3. Select Action | Make Addendum.

4. Enter the text of the addendum.

5. Select the Action menu and then one of the signing options that is available:

· Add to Signature List

· Save without Signature

· Sign Note Now

Note: Whether you have these options available depends on your level of authorization (key).
