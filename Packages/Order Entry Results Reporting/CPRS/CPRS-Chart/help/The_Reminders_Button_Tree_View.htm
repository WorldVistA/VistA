The Reminders Button Tree View

You get the view shown below when you click the Reminders button. The Reminders drawer has a similar tree view.

You cannot begin processing reminders from this dialog until you begin a new progress note. You can, however, review the reminders that are due and applicable to this patient. Coordinators can also use this dialog to quickly refresh dialogs as they build them.

 

The View menu contains four options:

·      Clinical Maintenance

·      Education Topic Definition

·      Reminder Inquiry

·      Reference information

 

The Action menu contains these options:

·      Evaluate Reminders

·      Evaluate Processed Reminders

·      Refresh Reminder Dialogs

 

CACs use the Refresh Reminder Dialogs option when they are creating the reminder dialogs. Then, they do not have to constantly exit the CPRS GUI and launch it again to see their changes.

To process reminders, you must have an unsaved progress note for the selected patient open. You cannot process reminders directly from the Available Reminders tree view that you get by clicking the Reminders button.

Select View on the Available Reminders dialog to determine or change the types of reminders that are displayed in the Reminders Drawer. A check mark next to the icon indicates that that type of reminder will be included in the tree view. Click on any of the menu items to change it.
