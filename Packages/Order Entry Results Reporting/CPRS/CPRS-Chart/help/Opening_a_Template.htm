Opening a Template

To open a shared template or personal template (if you have created any):

1.       Select the Notes tab.

2.      Select the Templates drawer.

Note:      You may open the Templates drawer without creating a new note, which gives you the ability to use templates in any CPRS text field if you use Copy Template Text.

 

3.      Expand either the Shared Template or Personal Templates trees.

4.      Right-click the desired template.

5.      From the resulting menu, select Preview/Print Template to see the template.
