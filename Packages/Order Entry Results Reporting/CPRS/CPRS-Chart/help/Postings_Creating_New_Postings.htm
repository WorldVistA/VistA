Creating New Postings

You create the following types of postings by creating progress notes using note titles that your site's IRM staff has configured for this purpose. (Check with your site's IRM staff if you don't know which note titles create which types of postings.)

路      Clinical Warning (which is the same as Warning)

路      Crisis Note

路      Directive

路      Warning

 

For example, to create a posting for a crisis note, take the following steps:

1.     Select the Notes tab.

2.     Select New Note.

CPRS displays the Progress Note Properties dialog.

 

3.     In the Progress Note Title pane, select a note title that generates a crisis note.

4.     In the Date/Time of Note field, select a date.

5.     In the Author field, select an author.

6.     Select OK.

7.     From the main menu, select File | Refresh Patient Information. CPRS displays the letter C (for crisis note) on the Postings button and, in the Postings pane on the Cover Sheet tab, displays the title Crisis Note and the date you selected for the note.

 

To create a posting for an allergy or adverse reaction, enter the allergy from either the Cover Sheet tab or the Orders tab. (See Entering Allergies from the Cover Sheet Tab and Entering Allergies from the Orders Tab, respectively.)

Note:     Although you may be able to enter progress notes for allergies/adverse reactions, doing so does not create Allergies postings. As mentioned above, you can create Allergies postings only by entering allergies via the Cover Sheet or Orders tab. CPRS cannot perform order checks on allergies you document via progress notes.
