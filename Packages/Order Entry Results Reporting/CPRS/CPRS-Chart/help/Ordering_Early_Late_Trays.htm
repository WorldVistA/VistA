Ordering Early/Late Trays

To order early or late trays, follow these steps:

  1. Select the Orders tab.
  2. Select Diet in the Write Orders list.

    Note:      The Encounter Information dialog appears if no encounter information has been entered.

  3. If there is a conflict, ensure that the order you are entering will not create a problem with a current or delayed diet order.
  4. Select the Early/Late Tray tab.
  5. Select the mean (Breakfast, Lunch, or Dinner).
  6. Select the time you want the tray delivered.
  7. Enter the Start Date/Time using the following steps:
    1. Select the calendar button in the Start Date/Time field.
    2. Choose the date. Arrows to the right and left of the month allow you to change the month. You then click on the desired day in the calendar shown.
    3. Choose a time. You can click on the hour and the minutes, choose Now, or choose midnight. Select OK.
  8. Enter the Stop Date/Time (use step a-c above except use the Stop date).
  9. Select the days of the week that you want the tray delivered at this time.
  10. Select Accept Order.
  11. When finished, select Quit.

Note:      The order must be signed before it is sent. You can either sign the order now or wait until later.
